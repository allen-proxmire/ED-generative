# Black-Hole — Folder Guide

**What this folder is.** The black-hole / horizon sector: the horizon as a substrate **decoupling surface** (the `b→0` locus), trans-Planckian resolution, no singularity, the Planck-mass remnant, the **area law**, the **Hawking spectrum**, greybody factors, the **Page curve**, and the resolution of the information paradox. The unifying object is the horizon itself — shown to be one substrate structure across black-hole, Rindler, cosmological, and acoustic cases.

**State** *(authoritative source: `event-density/docs/ED_Research_Targets.md`, target #4; PAPERS_INDEX)*:
- **Horizon thermodynamics is delivered:** the **area law `S ∝ A`** (form forced; coefficient inherited), the **Hawking spectrum `T_H = κ/2π`** (V5-KMS), and **`S = A/4` is now fully structural** (½ from `κ`, `2π` from the near-horizon Rindler profile — both ED-derived).
- **The Page-curve structure is reproduced** (Paper_050, via the V5 entanglement-bandwidth cap); the information paradox is audited as *not generated* (Paper_051).
- **Open:** **unitary completeness** — the Page-curve *structure* is there, but full unitarity is not yet shown (target #4). Also the merger-lag prediction's `τ_V5` floor (Paper_052.5) is open.
- **Cross-class:** Paper_047.5 shows BH / Rindler / cosmological / acoustic horizons are **one substrate object** (ties to SCBU).

**Tier key:** `Derived` (from primitives) · `Grounded` (conditional/structural) · `Inherited` (a value from measurement, e.g. the area-law coefficient, greybody factors) · `Prediction` (falsifiable) · `Synthesis` (arc consolidation) · `Open` (unitary completeness; the merger-lag floor).
