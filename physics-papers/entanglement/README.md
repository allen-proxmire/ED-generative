# Entanglement — Folder Guide

**What this folder is.** The entanglement sector: how quantum entanglement and its structure emerge from the substrate. Entanglement is read as the **unresolved-individuation regime** of participation, carried by the **cross-chain V5 kernel** (the correlation between distinct chains). The arc derives tensor-product non-factorizability, the Schmidt decomposition, monogamy, no-signaling, von Neumann entropy, the Bell-Tsirelson bound, and the ER=EPR-class echo.

- **No open frontier here** beyond the general cross-arc ones (the ER=EPR echo ties into the black-hole area-law arc; see `black-hole/` and target #4).
