# Scale Correspondence (ED-SC 4.x) — Folder Guide *(one boundary, six projections)*

**What this folder is.** The **ED-SC "Scale Correspondence" program** (the papers render it *Cross-Scale Invariance Extensions*). Its one claim: a family of otherwise-unrelated physical constants — the black-hole horizon `r_H = 2GM/c²`, the Hubble horizon `R_H = c/H₀`, the ED canonical operating point `ξ_canonical`, the MOND acceleration `a₀`, the quantum-computing coherence cap `𝓜_crit`, and the soft-matter hydrodynamic quality factor `Q ≈ 3.5` — are **not independent**. Each is a *projection of one substrate–cosmology boundary* (the surface `R_H = c/H₀` where the substrate's V5 cross-chain correlation budget saturates) onto a different parameter axis, reached by a different projection mechanism. In one phrase: **one boundary `R_H = c/H₀`, six projections, four RG regimes**, spanning ~40 orders of magnitude.

**State** *(this is an OPEN, falsifiable hypothesis program — NOT a closed derivation)*:
- **Every 4.x paper carries the same verdict: M3** — the correspondence *form* is FORCED, the numerical *values* are INHERITED from standard/empirical expressions. No paper reaches M2 (value-forced) or M1 (unconditional).
- **The whole arc is gated on ONE open derivation — "Route A" in [SC 4.2](Paper_ED_SC_4_2_xi_canonical_H0_derivation.md)** (the `ℓ_V5(H₀)` substrate–cosmology coupling). Closing Route A → simultaneous arc-wide M3→M2; closing Routes A+B+C with no new postulates → the corpus's first cross-arc M1. The post-2026-05-17 "Route A4" closure is heavily caveated: it closes only at *substrate-parameter-inherited* level, **conditional on the unproved reduction A-A4-4**, and `Θ_ED ≈ 10⁻¹²²` is inherited, not derived. (A 2026-07-06 note flags a `Θ_ED` primitive mislabel.)
- **The flagship falsifier is a genuine prediction:** synchronized co-variation of all six projections under Hubble-tension-class `H₀` shifts (SC 4.6 §5). If `a₀` tracks `H₀` but `Q` or `𝓜_crit` does not, the boundary-anchoring is refuted for that projection.
- **Honest negatives are on the record** — `r*` was *retired* (2026-04-23) from cross-scale invariant to a filter-dependent statistic (its `R² ≈ −1.88` is a negative goodness-of-fit, worse than null); GRF-Gaussianity is an unproven regime hypothesis "that may be false."
